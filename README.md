# ZenKanban ✨

> Un tableau Kanban élégant, premium et entièrement responsive, construit avec Vue 3 et Vite.

🌐 **[Tester l'Application en Ligne (Live Demo) 🚀](https://VOTRE_LIEN_PRECIS_ICI.com)**

ZenKanban réinvente le traditionnel tableau Kanban pour offrir l'expérience et le design d'un produit SaaS haut de gamme. L'application combine une esthétique moderne avec une productivité fonctionnelle pour proposer un environnement de travail apaisant et efficace.

## 🌟 Fonctionnalités

- **UI/UX Premium :** Magnifique design "Glassmorphism" avec un arrière-plan à effet parallaxe interactif et des transitions fluides.
- **Architecture Responsive :** L'interface s'adapte élégamment, passant de colonnes sur ordinateur à une disposition empilée sur mobile.
- **Drag & Drop Intelligent :** Déplacement naturel des tâches entre les colonnes à la souris (Drag & Drop HTML5 natif) sur ordinateur.
- **Contrôles Tactiles :** Des boutons intuitifs (← et →) permettent un déplacement des tâches particulièrement adapté aux écrans tactiles.
- **Édition Instantanée (Inline) :** Renommez vos tâches très simplement, au double-clic ou via une icône, directement depuis la carte du tableau.
- **Support du Dark Mode :** Adaptation automatique au thème de votre système (clair / sombre), complétée par un bouton manuel pour switcher à volonté.
- **Micro-interactions :** Animations douces et personnalisées, incluant une chute de pétales de cerisiers (Sakura) lors de l'accomplissement d'une tâche ! 🌸

## 🚀 Installation & Démarrage

Assurez-vous d'avoir installé [Node.js](https://nodejs.org/), puis suivez ces étapes :

1. Clonez ce repository
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
4. Compilez pour la production :
   ```bash
   npm run build
   ```

## 🛠 Technologies Utilisées
- Vue 3 (Composition API)
- Vite
- Vanilla CSS (Variables CSS personnalisées pour le Dark Mode)
- Web Animations API
